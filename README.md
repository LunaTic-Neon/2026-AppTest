# Ruin's City

Ruin's City는 폐허가 된 도시를 배경으로 한 탑다운 생존 슈팅 게임입니다. 플레이어는 몰려오는 적을 피해 움직이고, 자동 공격과 마우스 조준 공격을 활용해 오래 버티며 보스를 상대하게 됩니다.

이 프로젝트는 React UI와 HTML5 Canvas 기반 실시간 전투를 결합한 구조로 만들어져 있습니다. 메뉴와 설정은 React가 담당하고, 실제 게임 로직은 별도의 GameLoop에서 돌아갑니다.

## 게임 특징

- 탑다운 아레나 생존 슈터
- 마우스 조준 + 자동 공격 혼합 전투
- 레벨업 카드 선택 방식의 성장 시스템
- 중간보스와 최종보스가 포함된 생존 흐름
- 프롤로그와 스테이지별 스토리 대화
- 골드 기반 메타 진행
- 배속 해금 및 설정 기능

## 현재 들어있는 주요 콘텐츠

- 스토리 선택 화면
	- PROLOGUE
	- 1-1
	- 1-2
	- 1-3
- 프롤로그 초회 보상
	- 첫 완료 시 금화 500 지급
- 설정 기능
	- 마스터 / 배경음 / 효과음 / 보이스 볼륨
	- 볼륨 수치 직접 입력
	- 언어 선택: 한국어 / English / 日本語
	- 해금된 배속 선택
- 상점 기능
	- 배속 업그레이드 구매
	- 상위 배속은 이전 단계 구매 후 해금
- 진행 표시
	- 클리어한 스테이지 타일에 체크 표시

## 조작 방법

- 이동: WASD 또는 방향키
- 공격: 마우스 좌클릭
- 대시: Space
- 일시정지 / 복귀: Esc

## 플레이 흐름

1. 메인 메뉴에서 플레이를 선택합니다.
2. 프롤로그 또는 스테이지를 고릅니다.
3. 스토리 대화를 본 뒤 전투에 들어갑니다.
4. 전투 중 레벨업 카드를 선택하며 캐릭터를 강화합니다.
5. 중간보스와 최종보스를 넘기면 스테이지 클리어 화면으로 이동합니다.
6. 획득한 골드로 상점에서 배속을 해금할 수 있습니다.

## 기술 스택

- React 18
- TypeScript
- Vite
- Zustand
- Tailwind CSS
- HTML5 Canvas

## 실행 방법

```bash
npm install
npm run dev
```

개발 서버 실행 후 브라우저에서 기본 Vite 주소로 접속하면 됩니다.

## 빌드

```bash
npm run build
npm run preview
```

## 프로젝트 구조 간단 안내

- `src/App.tsx`: 메뉴, 설정, 상점, 스토리, GameLoop 연결
- `src/game/GameLoop.ts`: 실시간 게임 진행의 중심 로직
- `src/game/canvas/CanvasRenderer.ts`: 캔버스 렌더링
- `src/store/`: 플레이어, 적, 게임 진행도, 메타 진행도 상태 관리
- `src/components/`: 레벨업, 설정, 스토리, 스테이지 클리어 등 UI 컴포넌트
