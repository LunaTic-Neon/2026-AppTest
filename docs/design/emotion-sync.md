# Emotion Sync 시스템 + 엔딩 분기

## 개요

VN 선택지·이벤트 → **4종 감정 게이지** → 전투 보조 / Sync % 누적 → **엔딩 분기**

## 감정 타입

| ID | 이름 | VN에서 쌓이는 조건 | 전투 효과 | 모듈 연동 |
|----|------|-------------------|-----------|-----------|
| `joy` | 기쁨 | 카나데 긍정 이벤트, 소소한 승리 | 이동·회피 +15% | 도주: 경로 옵션 +1 |
| `anger` | 화남 | 억압·배신·위기 | 공격·범위 +20% | 보스: 광역기 해금 |
| `sadness` | 슬픔 | 상실·정체성 혼란 | 방어·지속 +25% | 카드: 방어/흡수 강화 |
| `love` | 사랑 (히든) | 로맨스 이벤트, Sync 고조 | Sync 가속 ×1.5 | 최종전 100% 조건 |

- 각 감정: **0~100** (표시는 0~10 또는 % 선택)
- `love`는 UI에 기본 비표시, 임계값 20 이상 시 실루엣 아이콘 등장

## Sync % 마일스톤 (스토리 고정)

| Sync | 시점 | 이벤트 |
|------|------|--------|
| 10% | 1장 | Emotion Sync 각성, 광역기 |
| 40% | 2장 | 닥터 제로 1차 업그레이드 |
| 70% | 3장 | 렌 구출 직후 |
| 100% | 5장 | 카나데와 마음 확인 → 최종전 |

Sync % = `(joy + anger + sadness + love×2) / 5` 기반 가중 평균 + 스토리 마일스톤 최솟값

## 카나데 Affection

- 0~100, 로맨스 엔딩 임계값: **70**
- 선택지 `affection_kanade` +5~15 / -5
- Sync 100% 직전 로맨스 이벤트: affection ≥ 70 필요

## 엔딩 3종

### 일반 엔딩 — 「폐허의 아침」

| 조건 | 내용 |
|------|------|
| 5장 클리어 | ✓ |
| `mentor_rescued` | ✓ (필수) |
| affection < 70 또는 love < 20 | — |

**결과:** 엘라·카나데·제로·렌이 아스포델에서 새 균형을 찾아 살아감. 장비는 봉인.

---

### 로맨스 엔딩 — 「같은 리듬」

| 조건 | 내용 |
|------|------|
| 일반 조건 | ✓ |
| `affection_kanade` ≥ 70 | ✓ |
| `love` ≥ 20 (히든 해금) | ✓ |
| 5장 직전 로맨스 이벤트 | ✓ |

**결과:** 엘라와 카나데의 관계에 초점. Sync 100%로 각성했으나 장비는 자발적 제한. 둘만의 일상.

---

### 진(真) 엔딩 — 「아스포델의 조율」

| 조건 | 내용 |
|------|------|
| 로맨스 조건 | ✓ |
| `met_vale` + 4장 대화에서 「대화」선택 | ✓ |
| 5장 최종 선택: 「모두를 위한 재편」 | ✓ |

**결과:** Director Vale와의 협상, 감정 AI의 윤리적 재배치. 적 포함 모두가 새 질서 아래 행복 — 카나데와 엘라, 렌·제로, 상층·하층의 화해.

---

## 멘토 구출

- **모든 엔딩의 전제.** 실패 시 재시도 (게임오버 없음).
- 3장 보스전 클리어 = `mentor_rescued = true`
