## 흐름 개요

```
프롤로그 → 인터루드(훈련+일상) → 1장(유메~Dr.제로) → 2장~5장
```

|------|-----|------|------|-----------|-----|

## 1장 상세 (현재 구현)

1. **Act 1** — 생필품 수색 중 유메 구출, 렌과 소개
2. **Act 2** — 유메의 전술·요리·고난 공유, 호감·일상
3. **Act 3** — 순 patrol 교전, Emotion Sync 10%, 유메가 음지 네트워크 조사
4. **Act 4** — Dr. 제로 연구실, 장비·사이보그 **암시**, 완전 해독은 후속

## 2장 이후 제안 (기획)

- **2장**: Dr. 제로 거주 연구실로 이동, 1차 업그레이드, 유메와 엘라 둘만의 밤 (로맨스 씨앗)
- **3장**: 장비 신호 추적 → 렌 납치 → 구출 보스전 (실패 없음)
- **4장**: Director Vale 대면, 제로의 과거, 진엔딩 복선
- **5장**: Sync 100%, 3종 엔딩

## 추가 연출 제안

1. **인터루드 CG/일러스트**: 「같은 매트리스 옆에서 잠드는」 장면 — 유메 등장 전 고독 대비
2. **1장 소소한 로맨스**: 유메가 엘라 손목 부상 치료 시도 → 엘라 회피 → 유메 「괜찮아, 천천히」
3. **Dr. 제로**: 첫 만남에서 엘라만 보고 「흥미롭네」 — 유메 질투/오해 1회
4. **장비**: Sync 10% 이후 평소에도 가끔 시스템 whisper — 불안감 유지
